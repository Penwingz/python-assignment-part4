# Python Assignment — Part 4: Data Visualization & Machine Learning

## Theme: Student Performance Analysis & Prediction

End-to-end data analysis: explore a student dataset with pandas, produce visualizations with Matplotlib and Seaborn, and build a Logistic Regression classifier to predict pass/fail.

## Tasks
- **Task 1** — Pandas exploration: head, shape, dtypes, describe, pass/fail counts, subject averages by outcome, top student
- **Task 2** — 5 Matplotlib plots: bar chart, histogram, scatter, box plot, line plot
- **Task 3** — 2 Seaborn plots: grouped bar chart (math/science), scatter with regression lines (attendance vs avg score)
- **Task 4** — Logistic Regression: train/test split, StandardScaler, training & test accuracy, per-student prediction report, feature coefficients chart
- **Bonus** — Predict pass/fail for a new student with `predict_proba`

## Files
- `part4_visualization_ml.ipynb` — main notebook (run with Kernel → Restart & Run All)
- `students.csv` — the dataset
- `plot1_bar.png` through `plot8_feature_importance.png` — all plot outputs

## How to run
Open `part4_visualization_ml.ipynb` in Jupyter, then go to **Kernel → Restart & Run All**.
